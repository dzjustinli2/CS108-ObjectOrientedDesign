1. User supplies an unsolved Sudoku puzzle.
	1.1. Not a valid puzzle.
	1.2. Throw an exception.
	1.3. Print an error message.
	1.4. Go back to step 1.

2. Print out the unsolved puzzle.

3. Solve the puzzle.
4. Store and print the first solution.
5. Store and print the number of solutions.
6. Store and print the time it took to solve the puzzle.

7. User makes changes to the unsolved puzzle thus generating a new puzzle. Go to step 1.
